# Settings UI

## Struktur

SettingsPanel
├── AudioTab
│   ├── Master Slider
│   ├── Music Slider
│   └── SFX Slider
├── VideoTab
│   ├── Fullscreen Toggle
│   ├── Resolution Dropdown
│   └── VSync Toggle
├── InputTab
│   └── Keybinding Liste
└── GameplayTab
    ├── Text Speed Dropdown
    └── Tutorial Toggle

## Verhalten
- Änderungen sofort übernehmen
- Optional: Apply/Cancel Button

## Integration
- Wird vom MenuManager geöffnet
