IT-AA 2D – Unity Game

IT-AA 2D ist ein modular aufgebautes 2D-Spielprojekt mit Unity. Der Fokus liegt auf einer sauberen, skalierbaren Architektur, klar getrennten Features und einer soliden Basis für die schrittweise Erweiterung des Spiels.

Über das Projekt

Das Projekt verfolgt einen strukturierten Ansatz für die Entwicklung eines 2D-Spiels in Unity. Systeme und Features werden getrennt aufgebaut, damit der Code wartbar, erweiterbar und nachvollziehbar bleibt.

Ziele des Projekts:

  • saubere Projektstruktur
  • klare Verantwortlichkeiten pro Feature
  • einfache Erweiterbarkeit
  • gute Wartbarkeit
  • lernfreundlicher Aufbau

Architektur

Das Projekt orientiert sich an einer featurebasierten Struktur. Systeme werden logisch getrennt organisiert, damit Gameplay, UI, NPCs, Savegame und Player-Funktionen unabhängig wachsen können.

Wichtige Prinzipien:

  • Separation of Concerns
  • Single Responsibility Principle
  • modulare Erweiterbarkeit
  • klare Ordner- und Dateibenennung
  • Wiederverwendbarkeit von Komponenten
