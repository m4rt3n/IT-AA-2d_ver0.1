# ⚙️ Settings

## Zweck
Der `Settings`-Ordner enthält projektweite Konfigurationen und Einstellungen.

## Inhalte
- Unity-Konfigurationen
- Projektweite Parameter
- Engine-bezogene Settings

## Verwendung
Diese Einstellungen beeinflussen das Verhalten des gesamten Spiels.

## Beispiele
- Input-Konfigurationen
- Rendering-Einstellungen
- Physik-Parameter

## Hinweis
Dieser Ordner enthält keine Gameplay-Logik, sondern nur Konfiguration.
